# Périmètre JÀNGAT V1

JÀNGAT V1 est une PWA mobile-first dédiée à la réactivation des compétences d’ingénierie pédagogique. Elle couvre l’inscription/connexion, l’onboarding, un diagnostic déterministe, un parcours progressif, les modules 1 et 2, la voix navigateur, la progression et la gamification.

Les modules 3 à 21 du premier cursus restent visibles comme feuille de route. Le parcours professionnel EduConcret complet est désormais disponible après le prérequis, avec 34 domaines, positionnements, évaluations, missions, preuves, portfolio privé, soutenance et documents de carrière. Le paiement, le réseau social, le portfolio public et le matching automatique d’emploi restent hors périmètre.

Le navigateur fournit un mode local persistant utilisable sans configuration. La production doit activer Supabase avec les variables documentées dans le README.
