# Modèle de contenu

Le parcours contient des sections, unités, leçons et étapes. Une étape porte un type (`content`, `vocabulary`, `quiz`, `open`), un objectif, le contenu et, selon le type, des choix, un corrigé ou une grille critériée.

Les contenus publiés possèdent une révision, un hash et un statut. Une évolution majeure crée une nouvelle révision. Les réponses ouvertes conservent la première version, l’auto-évaluation et la version révisée avec les statuts `draft`, `submitted`, `self_reviewed`, `revised`.

Le diagnostic répartit 15 questions dans sept domaines. Le score global est la moyenne entière des scores par domaine; aucun appel IA n’intervient.
